# Architecture Decision Records (ADRs)

## Overview

This directory contains Architecture Decision Records (ADRs) documenting significant architectural decisions made in the Multi-Agent Tour Guide System.

ADRs follow the format proposed by Michael Nygard and are a standard practice in MIT-level software engineering projects.

New ADRs should follow the template in [template.md](template.md).

## Status Definitions

- **Proposed**: Under discussion
- **Accepted**: Decision made and implemented
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
